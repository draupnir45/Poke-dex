# Poke-dex

##모바일 포켓몬스터 도감
모바일로 작동하는 포켓몬스터 도감을 만들어 보았습니다.

##버전정보
- ver 0.2 (2017. 3. 7.) : 디테일 뷰 들어갔을 때 설명을 읽어주도록 지원.
- ver 0.1 (2017. 3. 6.) : 1세대 및 2세대 도감정보와 공식 일러스트, 포켓몬 위키 링크 접속용 웹뷰.
